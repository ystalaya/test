git init - инициализация репозитория
git remote add origin ссылка на репозиторий - привязать локальное хранилище к репозиторию в github
git add . - добавить новый файлы
git status - проверка что изменилось и все ли занесено в новый коммит
git commit -m "сообщение что сделано"
git push -f origin master - внести изменения в удаленный репозиторий
